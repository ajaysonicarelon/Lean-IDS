# Lean IDS - Intelligent Design System

Welcome to **Lean IDS**, a comprehensive design system built for Carelon and Elevance Health applications. This system provides a unified set of components, design tokens, and guidelines to create consistent, accessible, and beautiful user experiences.

---

## 🎯 What is Lean IDS?

Lean IDS is a modern design system that combines:

- **React & Angular Components** - Production-ready, accessible UI components for both frameworks
- **Design Tokens** - Centralized design decisions (colors, spacing, typography)
- **Documentation** - Comprehensive guides and examples
- **Accessibility** - WCAG 2.1 AA compliant by default
- **Theming** - Support for multiple brand themes (Carelon, Elevance)

## 🚀 Getting Started

Choose your framework to get started:

### For React Developers

```bash
npm install @ajaysoni7832/lean-ids-components @ajaysoni7832/lean-ids-tokens
```

```tsx
import { ThemeProvider, Button, InputField } from '@ajaysoni7832/lean-ids-components';
import { theme } from '@ajaysoni7832/lean-ids-tokens';

function App() {
  return (
    <ThemeProvider theme={theme}>
      <InputField label="Email" type="email" />
      <Button variant="primary">Submit</Button>
    </ThemeProvider>
  );
}
```

**📚 Full Guide:** See "Getting Started → React Guide" in the sidebar

### For Angular Developers

```bash
npm install @ajaysoni7832/lean-ids-tokens
```

```typescript
import { Component } from '@angular/core';
import { InputFieldComponent } from './components/input-field';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [InputFieldComponent],
  template: `
    <lean-input-field
      label="Email"
      type="email"
    ></lean-input-field>
  `
})
export class AppComponent {}
```

**📚 Full Guide:** See "Getting Started → Angular Guide" in the sidebar
