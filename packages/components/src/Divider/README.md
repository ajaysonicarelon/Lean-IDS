# Divider Component

Visual separator between content.

## Quick Start

```tsx
import { Divider } from '@ajaysoni7832/lean-ids-components';
<Divider />
```

## API

| Prop | Type | Default |
|------|------|---------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `spacing` | `string` | `'16px'` |

## Examples

```tsx
<Divider />
<Divider orientation="vertical" />
<Divider spacing="24px" />
```

## Summary

Use Divider to separate content sections.
