## SVG Files

Place your Carelon brand SVG files here:

1. **carelon-logo.svg** - Full Carelon logo with text (120px width)
2. **carelon-symbol.svg** - Carelon symbol/icon only (26px square)

## File Naming Convention

- Use lowercase with hyphens
- Be descriptive and consistent
- Examples:
  - `carelon-logo.svg`
  - `carelon-symbol.svg`
  - `carelon-logo-white.svg` (if you have color variants)

## Usage

These SVG files will be imported in `Brand.tsx`:

```tsx
import CarelonLogo from './assets/carelon-logo.svg';
import CarelonSymbol from './assets/carelon-symbol.svg';
```
