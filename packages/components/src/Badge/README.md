# Badge Component

Status indicators and labels.

## Quick Start

```tsx
import { Badge } from '@ajaysoni7832/lean-ids-components';
<Badge label="Active" type="success" />
```

## API

| Prop | Type | Default |
|------|------|---------|
| `label` | `string` | - |
| `type` | `'success' \| 'error' \| 'warning' \| 'info'` | `'info'` |

## Examples

```tsx
<Badge label="Success" type="success" />
<Badge label="Error" type="error" />
<Badge label="Warning" type="warning" />
<Badge label="Info" type="info" />
```

## Summary

Use Badge for status indicators and labels.
