# About Lean IDS

## 🎨 Design System by Carelon Global Solutions

**Lean IDS** (Internal Design System) is a comprehensive design system created and maintained by the **Carelon Global Solutions UX Team** (CGS UX Team).

## 🎯 Our Mission

To provide a unified design language that:
- ✅ Ensures consistency across all products
- ✅ Accelerates development with reusable components

## 🚀 Getting Started

### For Developers

Install Lean IDS in your project:

```bash
npm install @ajaysoni7832/lean-ids-components @ajaysoni7832/lean-ids-tokens
```

```tsx
import { Button, PageLayout } from '@ajaysoni7832/lean-ids-components';
import { theme } from '@ajaysoni7832/lean-ids-tokens';
```

### For Designers

**Need access to Figma design files?**

If you're a designer and need access to the Lean IDS Figma library, please submit a request through our ServiceNow portal:

👉 **[Request Figma Account Access](https://elevancehealth.service-now.com/ess?id=sc_cat_item&table=sc_cat_item&sys_id=41ba46279796e9501fc7b2c71153af87&searchTerm=figma)**

Once approved, you'll have access to all Lean IDS design files, components, and design tokens in Figma.

## 🎨 Design Philosophy

### **Lean & Efficient**
Components are lightweight and performant, ensuring fast load times and smooth interactions.

### **Intelligent & Accessible**
Every component is built with accessibility in mind, meeting WCAG 2.1 AA standards.

### **Consistent & Scalable**
Design tokens ensure visual consistency across all products and platforms.
